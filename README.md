# Team Project Management System

A full-stack Team Project Management web application where admins can create projects, assign tasks, and team members can view and manage their assigned tasks.

---

## Live Demo

Deployed Project Link:  
[Click Here to Open Project]  (https://team-project-management-kappa.vercel.app)

## Demo Video

[![Watch the video](https://youtu.be/FDsTJ_1wOR0)


## Features

- Admin Authentication
- Member Authentication
- Create Projects
- Assign Tasks
- Task Management
- Role-Based Access
- Dashboard UI
- Responsive Design

## Installation

### Clone Repository

```bash
git clone https://github.com/somii-i/Team-Project-Management-.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

---

## Environment Variables

Create `.env` file inside backend folder:

```env
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
PORT=5000
```
